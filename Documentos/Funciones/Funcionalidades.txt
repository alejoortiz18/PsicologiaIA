REGISTRO USUARIO
-	El sistema solicitará datos personales(Nombre, email, Identificación, alias, celular)
-	Botón registrar, 
-	EL sistema valida que no exista ni la cedula ni el correo ya registrados

-	El sistema enviará correo al usuario para validar el correo registrado con botón o link de comprobar correo + un token valido por 1 hora
-	El enlace del correo enviará el token a una pagina de crear contraseña.

-	Sino este activo muestra mensaje de token inválido y solicita reenvio de token.
-	Tendrá una opción para volver al landing page

-	Si el token este activo muestra formulario para establecer contraseña
